# Rocket-League-sideswipe-Taller-1-FIUBA

Para compilar seguir las siguientes instrucciones
```shell
rm build -rf
mkdir build
cd build
cmake ..
make -j4
```

Para iniciar el servidor
```shell
./server <service>
```

Para iniciar el cliente
```shell
./client <hostname> <service>
```
